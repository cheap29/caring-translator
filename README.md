# Caring Translator 🎀

選択したテキストをわかりやすくギャル翻訳してくれる Chrome 拡張機能です！

## ✨ 機能

- **右クリックメニュー**: 選択したテキストを右クリックして「ギャル翻訳」を実行
- **Side Panel 表示**: 翻訳結果を右サイドバーに表示
- **ギャル語翻訳**: 専門的な内容を軽快でわかりやすいギャル語に変換
- **絵文字付き**: 感情豊かな絵文字で楽しく読める
- **比喩・例え**: 複雑な内容も身近な例えで理解しやすく

## 🚀 インストール方法

### 1. ダウンロード

このリポジトリをクローンまたはダウンロードしてください。

### 2. Chrome 拡張機能として読み込み

1. Chrome で `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」を ON にする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このプロジェクトのフォルダを選択

### 3. API キーの設定

1. 拡張機能のアイコンを右クリック → 「オプション」を選択
2. OpenAI API キーを入力して保存
   - [OpenAI API](https://platform.openai.com/api-keys) で API キーを取得

## 📖 使い方

1. **テキスト選択**: 翻訳したいテキストを選択
2. **右クリック**: コンテキストメニューから「ギャル翻訳」を選択
3. **結果確認**: 右サイドバーに翻訳結果が表示されます

## 🎯 翻訳例

**原文**:

```
The Federal Reserve raised interest rates by 0.25% today, marking the third consecutive increase this year.
```

**ギャル翻訳**:

```
今日、FRBが金利を0.25%上げたって話っぽい〜 📈
今年で3回目の連続上げやね！💸

要するに、お金を借りるのがちょっと大変になったってことやで〜
でも、インフレ対策としては必要なことらしいよ✨

マジで経済って複雑やね〜😅
でも、これで物価が落ち着いてくれるといいな〜🙏
```

## 🛠️ 技術仕様

- **Manifest Version**: 3
- **Permissions**:
  - `contextMenus`: 右クリックメニュー
  - `activeTab`: アクティブタブへのアクセス
  - `scripting`: ページスクリプト実行
  - `storage`: 設定保存
  - `sidePanel`: サイドパネル表示
- **API**: OpenAI GPT-4o-mini
- **UI**: シンプルなモダンデザイン

## 📁 ファイル構成

```
caring-translator/
├── manifest.json      # 拡張機能の設定
├── background.js      # バックグラウンド処理
├── sidepanel.html     # サイドパネルのHTML
├── sidepanel.js       # サイドパネルのJavaScript
├── options.html       # 設定画面のHTML
├── options.js         # 設定画面のJavaScript
└── icons/            # アイコンファイル
```

## 🔧 開発

### ローカル開発

1. ファイルを編集
2. `chrome://extensions/` で「更新」ボタンをクリック
3. 変更が反映されます

### デバッグ

- 拡張機能のアイコンを右クリック → 「検証」でデバッグ
- サイドパネルは別ウィンドウで開くので、個別にデバッグ可能
